# KBO 기록실 앱

키움 히어로즈 팬을 위한 KBO 경기/선수/팀 정보 앱 (프로토타입)

## 셋팅 방법

### 필요한 것
- Node.js 18 이상

### 설치 및 실행

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm start
```

브라우저에서 http://localhost:3000 접속하면 바로 확인 가능합니다.
폰으로 볼 때는 PC와 같은 WiFi에 연결된 상태에서
http://[PC의 IP주소]:3000 으로 접속하세요.

---

## 현재 구현된 기능

### 경기 탭
- 날짜별 경기 조회 (이전/이후 날짜 이동)
- 키움 경기 항상 맨 위 + 초록 테두리
- LIVE / 종료(스코어) / 예정 상태 표시
- 경기 카드 클릭 → 선발 투수 / 홈 라인업 / 원정 라인업 확인
- 그라운드뷰 (포지션별 선수 배치)

### 팀 탭
- 10개 팀 목록 (키움 최상단)
- 팀 클릭 → 포지션별 로스터 (투수 / 포수 / 내야수 / 외야수)
- 등번호순 보기
- **등록/말소** — 1군 등록 선수 / 말소·부상 선수 구분 표시

### 선수 검색 탭
- 검색 UI (데이터 연동 전 UI만)

### 순위 탭
- 2025 시즌 팀 순위
- 키움 강조 표시
